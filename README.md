# NumBase — Number System Calculator & Converter

![NumBase](https://img.shields.io/badge/Status-Active-success)

NumBase is a modern, feature-rich web application designed for computer science students, engineers, and math enthusiasts. It provides real-time number system conversion and arithmetic calculations wrapped in a stunning, interactive "liquid glassmorphism" user interface.

## ✨ Features

* **Real-Time Base Converter:** Instantly convert between Binary (Base 2), Octal (Base 8), Decimal (Base 10), Hexadecimal (Base 16), and BCD (Binary-Coded Decimal).
* **Live Arithmetic Calculator:** Perform addition, subtraction, multiplication, and division across different number systems simultaneously without needing to press "Calculate".
* **Interactive Bit Visualizer:** See the real-time 8-bit group representation of your converted numbers.
* **Liquid Glassmorphism UI:** A highly polished, ultra-transparent frosted glass aesthetic with subtle light reflections.
* **Interactive Background:** Features a responsive, mouse-tracking animated birds background powered by Vanta.js and Three.js.
* **Quick Info Chips:** Automatically calculates and displays decimal value, bit length, byte value, and parity (even/odd).

## 🚀 Technologies Used

* **HTML5 & CSS3:** Semantic markup with modern CSS Grid/Flexbox and custom properties (variables) for the glassmorphism theme.
* **Vanilla JavaScript (ES6+):** Handles all mathematical logic, base conversions, and live DOM updates without heavy frameworks.
* **Three.js & Vanta.js:** Drives the interactive, 3D bird particle background.
* **Google Fonts:** Space Mono and DM Sans.

## 🛠️ Installation & Usage

This project is completely client-side and requires no build tools, package managers, or server setup.

1.  Clone or download this repository.
2.  Locate the `index.html` file.
3.  Double-click `index.html` to open it in your preferred modern web browser (Chrome, Firefox, Safari, Edge).
*(Note: An active internet connection is required upon first load to fetch the Three.js, Vanta.js, and Google Fonts CDNs).*
